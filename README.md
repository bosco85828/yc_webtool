# yc_webtool
* Environment

Python 3.9.13


* Description 

透過 Flask 架設內部平台並且透過對接 api 讓以下平台的操作可以統一進行在內部頁面。

1. CDNW
2. YCCDN
3. JulyCDN 
4. Cnzz友盟網站統計
